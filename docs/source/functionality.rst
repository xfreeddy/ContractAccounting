Функциональность приложения
============================

Описание системы
----------------

«Контракт-Учёт» — десктопное приложение для автоматизации учёта
договоров и контрактов в малой компании.

Функции доступные пользователям
--------------------------------

**Авторизация**
   - Вход в систему по логину и паролю.
   - Разграничение прав: Администратор и Менеджер.

**Учёт договоров**
   - Добавление нового договора (номер, дата, сумма, предмет, контрагент).
   - Редактирование существующего договора.
   - Удаление договора (только для Администратора).
   - Прикрепление скан-копии документа.

**Справочники**
   - Ведение списка контрагентов.
   - Управление типами и статусами договоров.

**Поиск и фильтрация**
   - Поиск по номеру договора, контрагенту, дате, статусу.
   - Фильтрация по периоду и сумме.

**Контроль сроков**
   - Автоматический расчёт дат окончания договоров.
   - Визуальное выделение договоров, истекающих в ближайшие 30 дней.
   - Уведомления при входе в систему о критических сроках.

**Отчёты**
   - Реестр договоров за период.
   - Список договоров с истекающим сроком.
   - Сводка по контрагенту.
   - Экспорт отчётов в Excel и PDF.
