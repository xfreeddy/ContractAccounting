Архитектура приложения
======================

Общее описание
--------------

Приложение «Контракт-Учёт» реализовано в виде десктопного приложения
на Python с графическим интерфейсом PyQt6 и базой данных MS SQL Server.

Паттерн MVC
-----------

Приложение построено по паттерну MVC (Model-View-Controller):

- **Model** — классы для работы с базой данных (SQLAlchemy ORM).
  Хранят и обрабатывают данные о договорах, контрагентах, пользователях.
- **View** — экраны и формы PyQt6. Отображают данные пользователю
  и принимают ввод.
- **Controller** — бизнес-логика. Связывает Model и View,
  обрабатывает действия пользователя.

Основные компоненты
-------------------

- **main.py** — точка входа, запуск приложения.
- **main_window.py** — главное окно, навигация между разделами.
- **database/** — модели данных и подключение к MS SQL.
- **ui/** — формы и виджеты PyQt6.

Взаимодействие компонентов
--------------------------

1. Пользователь взаимодействует с интерфейсом (View).
2. View вызывает методы Controller.
3. Controller обращается к Model для получения или сохранения данных.
4. Model выполняет запрос к MS SQL Server и возвращает результат.
5. Controller передаёт данные обратно в View для отображения.
