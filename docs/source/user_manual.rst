Руководство пользователя
=========================

Запуск приложения
-----------------

1. Откройте файл ``ContractAccounting.exe`` или выполните команду::

      python main.py

2. Появится окно авторизации.

Вход в систему
--------------

1. Введите логин и пароль в соответствующие поля.
2. Нажмите кнопку **«Войти»**.
3. При неверном пароле система выведет сообщение об ошибке.

Роли пользователей
------------------

- **Администратор** — полный доступ: добавление, редактирование,
  удаление договоров, управление пользователями.
- **Менеджер** — добавление и редактирование своих договоров,
  просмотр реестра.

Работа с договорами
-------------------

**Добавление договора:**

1. Нажмите кнопку **«Добавить»** в главном окне.
2. Заполните все обязательные поля формы.
3. Нажмите **«Сохранить»**.

**Редактирование договора:**

1. Выберите договор в списке двойным кликом.
2. Внесите изменения в форме.
3. Нажмите **«Сохранить»**.

**Удаление договора:**

1. Выберите договор в списке.
2. Нажмите кнопку **«Удалить»** (доступно только Администратору).
3. Подтвердите удаление в диалоговом окне.

Поиск и фильтрация
------------------

1. Введите текст в строку поиска вверху окна.
2. Выберите фильтр: по дате, статусу или контрагенту.
3. Нажмите **«Найти»**.

Типовые проблемы
----------------

**Приложение не запускается:**
   Убедитесь что установлен Python 3.9+ и все зависимости
   (``pip install -r requirements.txt``).

**Нет подключения к базе данных:**
   Проверьте настройки подключения к MS SQL Server в файле
   ``config.py`` (хост, порт, имя базы данных).

**Забыт пароль:**
   Обратитесь к Администратору системы для сброса пароля.
