Примеры использования
=====================

Сценарий 1: Добавление нового договора
---------------------------------------

1. Запустите приложение и войдите в систему.
2. В главном меню нажмите кнопку **«Добавить договор»**.
3. Заполните поля формы:

   - Номер договора: например, ``№ 25/06-001``
   - Дата заключения: выберите дату в календаре
   - Контрагент: выберите из списка или добавьте нового
   - Предмет договора: краткое описание
   - Сумма: укажите сумму в рублях
   - Дата окончания: укажите срок действия договора

4. При необходимости прикрепите скан-копию через кнопку **«Прикрепить файл»**.
5. Нажмите **«Сохранить»** — договор появится в общем реестре.

Сценарий 2: Поиск договора по контрагенту
------------------------------------------

1. В главном окне перейдите в раздел **«Реестр договоров»**.
2. В строке поиска введите название или ИНН контрагента.
3. Нажмите **«Найти»** — система отобразит все договоры с данным контрагентом.
4. Нажмите на нужный договор для просмотра подробной информации.

Сценарий 3: Просмотр договоров с истекающим сроком
----------------------------------------------------

1. При входе в систему появится уведомление о договорах,
   срок которых истекает в ближайшие 30 дней.
2. Перейдите в раздел **«Контроль сроков»**.
3. Договоры с истекающим сроком выделены красным цветом.
4. Выберите договор и нажмите **«Редактировать»** для продления срока.
