# Инструкция по запуску и настройке АС «Контракт-Учёт»

## 1. Назначение

АС «Контракт-Учёт» — десктопное приложение для учёта договоров и контрактов
малой компании. Поддерживает разграничение прав (Администратор, Менеджер),
реестр договоров с поиском и фильтрацией, контроль сроков действия.

## 2. Системные требования

- ОС: Windows 10 / 11
- MS SQL Server 2019+ (Express или выше)
- ODBC Driver 17 for SQL Server
- (только для запуска из исходников) Python 3.10+

---

## 3. Вариант А. Запуск исполняемого файла (.exe) — для конечного пользователя

Самый простой способ. Python устанавливать **не нужно**.

1. Скопируйте файл `dist\ContractAccounting.exe` на рабочий компьютер.
2. Убедитесь, что на компьютере доступен MS SQL Server с базой `УчётДоговоров2`
   (см. раздел 5 «Настройка базы данных»).
3. Дважды кликните по `ContractAccounting.exe`.
4. В окне авторизации введите тестовые учётные данные:
   - Администратор: `admin` / `123`
   - Менеджер: `manager` / `123`

---

## 4. Вариант Б. Запуск из исходного кода (для разработчика)

1. Установите Python 3.10+ и ODBC Driver 17.
2. Создайте виртуальное окружение:
   ```
   python -m venv venv
   ```
3. Активируйте его:
   ```
   venv\Scripts\activate
   ```
4. Установите зависимости:
   ```
   pip install -r requirements.txt
   ```
5. Создайте базу данных (раздел 5).
6. Запустите приложение:
   ```
   python main.py
   ```

---

## 5. Настройка базы данных

1. Откройте **SQL Server Management Studio (SSMS)**.
2. Подключитесь к своему серверу (например, `localhost\SQLEXPRESS`).
3. Откройте и выполните скрипт `setup_db.sql` — он создаёт базу,
   таблицы (Users, Counterparties, Contracts), триггер обновления статуса
   и тестовые данные.
4. При необходимости измените строку подключения в `database/repository.py`
   (имя сервера, имя базы).

---

## 6. Сборка исполняемого файла (.exe)

Сборка выполняется один раз разработчиком:

1. Активируйте виртуальное окружение.
2. Установите PyInstaller: `pip install pyinstaller`
3. Запустите сборку:
   ```
   build.bat
   ```
   либо вручную:
   ```
   pyinstaller --onefile --windowed --name "ContractAccounting" main.py
   ```
4. Готовый файл появится в папке `dist\ContractAccounting.exe`.

## 7. Устранение возможных проблем (Troubleshooting)

| Проблема | Причина | Решение |
|----------|---------|---------|
| При запуске: «Не удалось подключиться к БД» | Не запущен MS SQL Server или неверная строка подключения | Запустите службу SQL Server; проверьте имя сервера и базы в `database/repository.py` |
| «Data source name not found» / ошибка ODBC | Не установлен ODBC Driver 17 | Скачайте и установите *ODBC Driver 17 for SQL Server* с сайта Microsoft |
| Окно входа открывается, но логин не проходит | В базе нет тестовых пользователей | Повторно выполните `setup_db.sql` (раздел вставки данных в Users) |
| `.exe` не запускается, мелькает окно и пропадает | Отсутствует подключение к БД или антивирус блокирует | Запустите из командной строки, чтобы увидеть ошибку; добавьте файл в исключения антивируса |
| `ModuleNotFoundError` при запуске из исходников | Не активировано venv или не установлены зависимости | `venv\Scripts\activate`, затем `pip install -r requirements.txt` |
| Долгий первый запуск `.exe` | Распаковка onefile-сборки во временную папку | Это нормально для PyInstaller `--onefile`; последующие запуски быстрее |
